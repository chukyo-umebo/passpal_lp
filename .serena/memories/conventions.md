# Code Style and Conventions

**File Structure**:
- **App Router**: `src/app`.
- **Components**:
    - UI: `src/components/ui` (Shadcn).
    - Feature: `src/components`.
- **Hooks**: `src/hooks`.
- **Utils**: `src/lib/utils.ts`.

**Naming**:
- **Components**: PascalCase (e.g., `CTASection.tsx`).
- **Utils/Hooks**: camelCase/kebab-case (e.g., `use-mobile.ts`).

**Style**:
- **TypeScript**: Strict mode.
- **Imports**: Use `@/` alias.
- **Styling**: Tailwind CSS.
- **Icons**: `lucide-react`.

**Library**:
- **Shadcn/ui**: "New York" style.
