# Project Summary

**Project Name**: PassPal LP (Landing Page)

**Purpose**:
This project is the landing page for "PassPal", a comprehensive utility application designed for students. The app aims to help students manage their academic life, including:
- **Smart Timetable**: Visual timetable management with widget support.
- **Notifications**: Reminders for classes and assignments (planned).
- **Material Management**: Organizing lecture notes and materials (planned).
- **Grade Tracking**: GPA calculation and credit tracking (planned).
- **Assignment/Exam Management**: Tracking deadlines and schedules (planned).

**Current Status**:
The landing page showcases the app's features, some of which are currently marked as "Not Implemented" (未実装). It includes sections for features, screenshots, and likely download links.

**Target Audience**:
Students (University/College).
